Os principais estilos de percussão brasilero de mão.

# Ijexa

O toque basico:
```
 ₁... ₂... ₃... ₄...
|Ṣ---|Ṣ-O-|Ṣ---|Ṣ-O-|
```

O toque com síncopa:
```
 ₁... ₂... ₃... ₄...
|O---|O---|O-O-|O---|
|-O--|O--O|O-O-|O---|
|SS--|--OO|O-O-|O---|
```

# Barravento

Cado compasso contém uma tercina.
```
 ₁.. ₂.. ₃.. ₄..
|OSS|SOO|OSS|SOO|
 ₁.. ₂.. ₃.. ₄..
|O-S|SOO|OSS|SOO|
 ₁.. ₂.. ₃.. ₄..
|O-S|S-O|O-S|S-O|
```

# Samba

```
 ₁... ₂... ₃... ₄...
|O--O|O--O|O--O|O--O|
 ₁... ₂... ₃... ₄...
|O--S|O--O|O--S|OOOO|
 ₁... ₂... ₃... ₄...
|OS-S|O--O|O--S|OOOO|
```

As variações na quarto compasso:

```
    ₁... ₂... ₃... ₄...
1: |OS-S|O--O|O--S|OOOO|
2: |OS-S|O--O|O--S|POOO|
3: |OS-S|O--O|O--S|OO-O|
```
